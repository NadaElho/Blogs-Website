## Run Locally

Clone the project

```bash
  git clone https://github.com/NadaElho/Blogs-Website.git
```

Go to the react project directory

```bash
  cd "Blog website"
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
